# iSCSI Examples

These examples use `iscsi.csi.windows.microsoft.com` and
`StorageClass/iscsi-for-windows-rwo`.

## Order

1. `01-storageclass`
2. `02-filesystem-pvc-pod`
3. `03-snapshots`
4. `04-restore-from-snapshot`
5. `05-expand-online`
6. `06-raw-block`
7. `07-static-pv-import`

iSCSI supports RWO filesystem volumes, raw block volumes, CHAP secrets,
snapshots, restore, expansion, and static imports.
